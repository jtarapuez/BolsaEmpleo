Registro de Postulante - Bolsa de Empleo

Paso 1: Información Básica

En este paso se registrarán los datos personales del postulante.

Tabla: boe_aspirante_tbl

Los campos de este paso incluyen:

  • Número de cédula
  • Fecha de emisión
  • Nombres y apellidos
  • Correo electrónico
  • Teléfono
  • Dirección
  • Otros datos personales

Paso 2: Formación Académica

En este paso se registrará la formación académica del postulante.

Tabla: boe_informacion_academica_tbl

Los campos de este paso incluyen:

  • Nivel educativo
  • Institución
  • Título obtenido
  • Área de estudio
  • Fechas de inicio y fin
  • Estado del estudio

Paso 3: Experiencia Laboral

En este paso se registrará la experiencia laboral del postulante.

Tabla: boe_antecedentes_laborales_tbl

Los campos de este paso incluyen:

  • Nombre de la empresa
  • Cargo desempeñado
  • Área de trabajo
  • Funciones realizadas
  • Fechas de inicio y fin
  • Referencias laborales

Paso 4: Cursos y Conocimientos

En este paso se registrarán los cursos y conocimientos adicionales del postulante.

Tabla: boe_conocimientos_adicionales_tbl

Los campos de este paso incluyen:

  • Nombre del curso
  • Institución
  • Tipo de curso
  • Área de estudio
  • Horas de duración
  • Certificado obtenido

label.style.opacity = '1'; }); }); } // Función para actualizar la línea de progreso según el paso activo del wizard function updateWizardProgressLine() { var wizardNav = document.querySelector('.custom-wizard-stepper .ui-wizard-nav'); if (wizardNav) { var activeStep = wizardNav.querySelector('li.ui-state-active'); if (activeStep) { var stepIndex = Array.from(wizardNav.children).indexOf(activeStep); var porcentaje = (stepIndex / (wizardNav.children.length - 1)) * 100; }); } // Usar MutationObserver como respaldo if (window.MutationObserver) { var observer = new MutationObserver(function(mutations) { setTimeout(initStepper, 200); }); observer.observe(document.body, { childList: true, subtree: true });
